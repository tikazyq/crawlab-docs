下载结果

下载结果

结果储存在数据库中之后，我们有时候需要将其导出，这时可以在界面中进行导出操作。

导航至任务详情，点击结果标签，点击下载CSV按钮，等待一会儿，结果就会以CSV的形式下载到本地。
